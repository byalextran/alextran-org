## Layout Primitives

| Primitive | Purpose |
|-----------|---------|
| [Stack](every-layout/stack.md) | Vertical spacing between sibling elements |
| [Box](every-layout/box.md) | Consistent padding and borders for containers |
| [Center](every-layout/center.md) | Horizontal centering with max-width constraints |
| [Cluster](every-layout/cluster.md) | Flexible, wrapping horizontal groups (tags, nav, buttons) |
| [Sidebar](every-layout/sidebar.md) | Two-element layout that stacks or sits side-by-side |
| [Switcher](every-layout/switcher.md) | Switches between row and column based on container width |
| [Cover](every-layout/cover.md) | Full-viewport sections with vertically centered content |
| [Grid](every-layout/grid.md) | Auto-flowing responsive columns |
| [Frame](every-layout/frame.md) | Aspect ratio containers for media |
| [Reel](every-layout/reel.md) | Horizontal scrolling for galleries/carousels |
| [Imposter](every-layout/imposter.md) | Absolutely-positioned overlays (modals, tooltips) |
| [Icon](every-layout/icon.md) | Icon-text pairing with proper alignment |
| [Container](every-layout/container.md) | Page-level width constraints and centering |
